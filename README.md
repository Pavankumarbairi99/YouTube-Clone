  
# YouTube Clone

A responsive **YouTube Clone** built using **React.js** with Redux for state management. Users can watch videos, post live comments, and enjoy a dynamic video streaming experience similar to YouTube.

## Features
  
- **Live Comments:** Users can post live comments on videos in real-time.  
- **User State Management:** Redux is used to store user cache for persistent state across pages.  
- **YouTube API Integration:** Fetch video data using YouTube API via GET requests.  
- **Responsive Design:** Works on desktop and mobile devices.  

## Tech Stack

- **Frontend:** React.js, Redux, CSS  
- **State Management:** Redux for caching user data  
- **API:** YouTube API for fetching video data
